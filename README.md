# GodotDemo
A simple platformer game with three levels.
Consists of :

<ul>
<li>Player with attached camera</li>
<li>Lives</li>
<li>Levels</li>
<li>Score</li>
<li>Simple AI</li>
<li>Menu System for Pause, New Game and win/lose conditions.</li>
</ul>
